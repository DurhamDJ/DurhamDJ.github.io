Livestream Widgets

On this page is a set of widgets you can use on DJ livestreams. To use a widget, just create a new browser source in your broadcasting software of choice with the given URL, width and height for that widget.

They are all large by default to make sure they look good at any reasonable size - to scale them down, use the transform options in your broadcasting software. They look best at 60fps, but should also look fine if you're streaming at other framerates.

Logo Spinner

  • URL: https://DurhamDJ.github.io/widgets/logo
  • Dimensions: 540x540
  • Parameters: interval

This widget rotates at a regular interval. By default this interval is 60 seconds, but here it is set to 5 seconds for demonstration purposes.

The interval can be customised by adding a parameter called interval to its URL. This should be given the value of the number of seconds it should wait between spins.

For example, to get a 30 second interval, the end of the URL would look like .../logo?interval=30.

DJ Name Tag

  • URL: https://DurhamDJ.github.io/widgets/nametag
  • Dimensions: ?x540
  • Parameters: name, interval

This widget features the same logo spinner as above, but adds text intended for use as a name tag.

The width of the widget is decided by the length of the name used, so we advise that you try increasing values until the full width of the widget is visible. As an example, the above widget has a width of around 2100 pixels.

There is a faint shadow around the name tag - we suggest that you err on the side of too much width, because you might cut it off if you aren't careful. For reference, the shadow has a spread of 20 pixels outside the visible name tag area.

Alongside the interval parameter defined above, this widget also offers name to allow you to specify your DJ name. The above demo uses these values: .../nametag?name=DJ Gerald&interval=5. If you find that putting a space in breaks things, try replacing spaces with %20 instead.
