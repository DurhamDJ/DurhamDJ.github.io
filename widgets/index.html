Livestream Widgets

On this page is a set of widgets you can use on DJ livestreams. To use a widget, just create a new browser source in your broadcasting software of choice with the given width and height for that widget.

They are all large by default to make sure they look good at any size - to scale them down, use the transform options in your broadcasting software.

Logo Spinner

https://DurhamDJ.github.io/widgets/logo_animation

This widget rotates at a regular interval. By default this interval is 60 seconds, but here it is set to 5 seconds for demonstration purposes.

The interval can be customised by adding a parameter called interval to its URL. This should be given the value of the number of seconds it should wait between spins.

For example, to get a 30 second interval, the end of the URL would look like .../logo_animation?interval=30.
